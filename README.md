Excited to share that we have successfully **developed and deployed** our full-stack project **Faculty Hub** 🌐

🔗 **Live Demo:** https://faculty-hub-iota.vercel.app

---

🔹 **What is Faculty Hub?**
A smart dashboard designed for **real-time faculty availability tracking** at MIT School of Computing. It helps users quickly find whether a faculty member is available, in class, or on leave.

---

🔹 **Key Features**
✅ Live Dashboard with faculty statistics
✅ Faculty Directory for easy search
✅ Floor-wise distribution of faculty
✅ Real-time status tracking (Available / On Leave / etc.)
✅ Clean and responsive UI

---

🔹 **Tech Stack Used**
⚛️ React + TypeScript
⚡ Vite (fast build tool)
🎨 Tailwind CSS
🗄️ Prisma (Database ORM)
🌐 Deployment on Vercel
