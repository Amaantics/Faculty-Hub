## 📖 Overview

**Faculty Hub** is a full-stack web application built for the MIT School of Computing that provides a clean, fast, and accessible interface for browsing faculty profiles, departments, and research areas. Designed with a modern UI powered by React and Tailwind CSS, and deployed with zero-config on both Vercel and Railway.

---

## ✨ Features

- 🔍 **Browse Faculty** — Explore profiles with research interests and department info
- 🏛️ **Department Directory** — Navigate faculty by school or department
- ⚡ **Blazing Fast** — Built on Vite for instant load times
- 📱 **Fully Responsive** — Works seamlessly on desktop and mobile
- 🎨 **Clean UI** — Minimal, modern design with Tailwind CSS

## 🚀 Getting Started

### Prerequisites
- Node.js **v18+**
- npm

### Installation

```bash
# 1. Clone the repo
git clone https://github.com/Amaantics/Faculty-Hub.git
cd Faculty-Hub

# 2. Install dependencies
npm install

# 3. Start dev server
npm run dev
```

Open **http://localhost:5173** in your browser.
